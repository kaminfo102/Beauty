# راهنمای تنظیم EmailJS برای ارسال ایمیل

## مراحل تنظیم

### 1. ایجاد حساب کاربری در EmailJS
- به [EmailJS.com](https://www.emailjs.com/) بروید و حساب کاربری رایگان ایجاد کنید

### 2. اتصال Gmail
- در پنل EmailJS: **Email Services** > **Add New Service** > **Gmail**
- با حساب Gmail خود وارد شوید
- **Service ID** را کپی کنید (مثلاً: `service_xxxxxxx`)

### 3. ایجاد Template ایمیل
- در پنل EmailJS: **Email Templates** > **Create New Template**

**موضوع ایمیل:**
```
درخواست جدید رزرو نوبت - {{clinic_name}}
```

**محتوای Template:**
```

- **Template ID** را کپی کنید (مثلاً: `template_xxxxxxx`)

### 4. دریافت Public Key
- در پنل EmailJS: **Settings** > **API Keys**
- **Public Key** را کپی کنید

### 5. تنظیم متغیرهای محیطی
1. یک فایل `.env.local` در ریشه پروژه ایجاد کنید
2. مقادیر زیر را اضافه کنید:

```env
NEXT_PUBLIC_EMAILJS_SERVICE_ID=service_xxxxxxx
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=template_xxxxxxx
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
```

### 6. تست
- سرور را اجرا کنید: `npm run dev`
- فرم را پر کنید و ارسال کنید
- بررسی کنید که ایمیل به صندوق ورودی شما ارسال شده باشد

## نکات مهم
- در پلن رایگان EmailJS، محدودیت 200 ایمیل در ماه وجود دارد
- فایل `.env.local` را در `.gitignore` قرار دهید
- برای استفاده در production، بهتر است از پلن پولی استفاده کنید
