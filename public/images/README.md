# تصاویر پروژه

این پوشه برای ذخیره تصاویر محلی پروژه است.

## لوگو:

- `logo.png` یا `logo.jpg` - لوگوی اصلی کلینیک (برای استفاده در Navbar)
  - فرمت توصیه شده: PNG با پس‌زمینه شفاف یا JPG
  - ابعاد توصیه شده: 512x512px یا 1024x1024px (مربع)
  - لوگو در وسط navbar قرار می‌گیرد و قسمتی از آن از navbar خارج می‌شود

## نام‌گذاری تصاویر:

### Hero Section:
- `hero-main.jpg` یا `hero-main.webp` - تصویر اصلی صفحه اصلی (800x500 توصیه می‌شود)

### Services Section:
- `service-laser.jpg` - لیزر موهای زائد
- `service-nutrition.jpg` - مشاور تغذیه
- `service-body.jpg` - دستگاه لاغری
- `service-injection.jpg` - تزریق ژل و فیلر

### About Section:
- `about-clinic.jpg` - تصویر کلینیک
- `about-equipment.jpg` - تجهیزات
- `about-staff.jpg` - متخصصان
- `about-services.jpg` - خدمات

### Gallery Section:
- `gallery-1.jpg` تا `gallery-6.jpg` - تصاویر گالری

## فرمت‌های توصیه شده:
- **WebP**: بهترین کیفیت و کمترین حجم
- **JPG**: سازگار با همه مرورگرها
- **PNG**: برای تصاویر با شفافیت

## ابعاد توصیه شده:
- Hero: 800x500px
- Services: 600x400px
- About: 600x400px (یا 600x300px)
- Gallery: 600x600px (مربع)

## نکات:
- تصاویر را با کیفیت مناسب ذخیره کنید (نه خیلی بزرگ که سرعت لود را کاهش دهد)
- از ابزارهای بهینه‌سازی تصویر استفاده کنید
- نام فایل‌ها را با حروف کوچک و خط تیره استفاده کنید
